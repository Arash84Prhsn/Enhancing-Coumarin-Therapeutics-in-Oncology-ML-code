# ML work for "Machine Learning-Guided Dose-Time Optimization and Experimental Validation Enhance Coumarin Therapeutics in Oncology"

> ***Please note that this repository is only meant for the reviewer and will be closed once the paper is published.(or at the very least the data will be removed)***

Within this github repository you shall find the ML implementations for this paper as well as the 
code used for generating some figures.

Everything is in the `main.ipynb`  notebook, but should the reviewer desire to use python files instead,
they can check `src` folder for .py files.

Should the reviewer intend to redo the gridSearching of each model, they can change the value of the
`DO_GRIDSEARCH` variable to `True` in each file or notebook cell.
